The Gantry — Layout Options

Full 16:9 overlay with standings, incidents, and lap graph. Which arrangement works best?

A — Standings Left, Right Split

Wide standings take up ~60% of the screen. Incidents and lap graph stack vertically on the right. Good for monitoring many drivers while watching incidents unfold.

B — True 3-Column
Standings
all drivers by class
Incidents
scrollable list
Lap Graph
gap chart

All three panels side by side. Incidents and lap graph each get their own full-height column. Maximizes vertical space for each section, best for tracking many incidents.

C — Standings Top, Panels Below
Standings — full width
all drivers by class, horizontal scroll
Incidents
Lap Graph

Standings span full width on top — maximizes horizontal space for all columns. Incidents and lap graph share the bottom half. Best column visibility, but less vertical room for incidents.

D — Tabbed / Switchable Panels
Standings
Incidents
Lap Graph
Active panel (full screen)

Tab bar across the top switches between full-screen views. Each panel gets the entire space. Best for focused workflows — but you can only see one panel at a time.
